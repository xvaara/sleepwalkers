# useCssSupports

SSR compatible and reactive .

**Package:** `@vueuse/core`
**Category:** Browser

## Usage

```ts
import { useCssSupports } from '@vueuse/core'

const { isSupported } = useCssSupports('container-type', 'scroll-state')
```

## Returns

| Name | Type |
| --- | --- |
| isSupported | `Ref` |
